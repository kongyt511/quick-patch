## Extension Settings
This extension contributes the following settings:

* `quick_patch.adb_path`: adb工具路径
* `quick_patch.dest_path`: 补丁推送的目的路径
* `quick_patch.patch_dir`: 补丁根目录(默认是debug)
* `quick_patch.scripts_dir`: 脚本根目录(默认是Scripts)
